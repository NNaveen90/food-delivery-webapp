# Food Delivery Web Application

A full-stack Food Delivery Web Application built using Java Servlets, JSP, JDBC, and MySQL.

## Tech Stack
- Java, Servlets, JSP, JSTL
- JDBC, MySQL
- MVC Architecture, DAO Pattern
- Apache Tomcat 11

## Features
- Customer registration and login
- Browse restaurants and menus
- Add to cart and place orders
- Real-time order status tracking
- Admin dashboard for managing restaurants, menus, orders and users

## Database
Run `sql/fooddelivery.sql` in MySQL Workbench to set up the database.

## How to Run
1. Import project into Eclipse as Maven project
2. Run `fooddelivery.sql` in MySQL
3. Update DB password in `DBConnection.java`
4. Run on Apache Tomcat 11
5. Open `http://localhost:8080/fooddelivery/`
